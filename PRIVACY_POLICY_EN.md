We use your data for:

## 5. Data Protection

### Security Standards

- **TLS/SSL** - Encryption for all data in transit
- **AES-256** - Encryption for sensitive data at rest
- **HTTPS** - All communications over secure protocol

### Security Measures

- Multi-factor authentication (MFA)
- Strong user verification
- Strict access control policies
- Regular security audits
- Automatic security updates

### Local & Cloud Storage

- Local data protected by Shared Preferences
- All cloud data encrypted in Firebase
- Automatic regular backups
- Users control local data deletion

## 6. Your Rights

You have the right to:

- **Access** - Request and download a copy of your data at any time
- **Correction** - Edit your personal information directly in the app
- **Deletion** - Delete your account and all associated data permanently
- **Portability** - Obtain your data in a portable format
- **Object** - Oppose processing of your data at any time
- **Withdraw Consent** - Withdraw permission for data use anytime

*Note: Some rights may be limited by legal requirements.*

## 7. Data Retention

| Data Type | Retention Period |
|-----------|-----------------|
